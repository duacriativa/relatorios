Como gerar o token do cliente

Feito uma única vez por cliente — leva 5 minutos.

1
Acesse o Graph API Explorer

Vá em developers.facebook.com/tools/explorer logado com o usuário que tem acesso admin à conta do cliente.

2
Selecione ou crie um App

No seletor "Meta App", escolha um app existente ou crie um em developers.facebook.com/apps (tipo Business).

3
Adicione as permissões

Clique em "Add a Permission" e adicione: ads_read, ads_management, read_insights. São somente leitura — não alteram nada nas campanhas.

4
Gere e copie o token

Clique em "Generate Access Token", autorize as permissões e copie. O token começa com EAA...

5
Encontre o Ad Account ID

No Gerenciador de Anúncios, o ID aparece na URL. Formato: act_123456789

6
Sobre validade do token

Tokens de usuário expiram em ~60 dias. Para uso permanente, gere um token de sistema no Business Manager: Configurações → Usuários do sistema → Gerar token.

Gerenciar Clientes

Clientes salvos no Supabase. Cadastre uma vez, use em qualquer computador.
